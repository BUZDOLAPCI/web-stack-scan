# Dedalus Marketplace Info

## Short Description
Fingerprint website technology stacks from headers and HTML

## Long Description
Detect web frameworks, CMSs, and libraries from HTTP headers and HTML signatures. Identify JavaScript frameworks, CSS frameworks, analytics scripts, and tracking pixels. Tools: fingerprint, detect_frameworks, detect_analytics.
